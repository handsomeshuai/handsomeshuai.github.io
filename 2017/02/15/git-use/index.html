4基本使用

  • git init
    • 初始化一个 Git 仓库
  • git status
    • 查看当前工作区、暂存区、本地仓库的状态
  • git add
  • git commit
    • 示例：git commit -m "日志说明" --author="操作者姓名 <邮箱>"
    • 执行 git commit 的时候，Git 会要求具有用户名和邮箱的参数选项
    • 可以通过 git config 命令配置一下用户名和邮箱
  • git log
  • gitk

总结：操作 Git 的基本工作流程就是先修改文件，然后执行 git add 命令。
git add 命令会把文件加入到暂存区，接着就可以执行 git commit 命令，将文件存入文档库，
从而形成一次历史记录。
